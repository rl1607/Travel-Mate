This is a starter MERN app scaffold matching the UI in the screenshots.

## Structure
- backend: Express + MongoDB
- client: Vite + React

## Quick start (locally)
1. Start MongoDB locally (or use MongoDB Atlas)
2. Backend:
   - cd backend
   - npm install
   - npm start
3. Frontend:
   - cd client
   - npm install
   - npm start
4. Open http://localhost:5173 and register/login.

Notes:
- API base URL set via VITE_API_URL environment variable in client.
- File uploads are stored in `backend/uploads/`.
